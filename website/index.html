Apps that live in a single file

It's just a web page

Plain HTML, the same stuff every website is made of. Rename it .html and it opens in any browser. HTML Clay only adds what a web page can't do alone: it saves your edits back into the file.

What people make

To-do list Journal Budget tracker Recipe box Habit tracker Reading list Personal wiki A little game

A file you own

One file, start to finish

Email it, back it up, drop it in any folder. No account, no cloud, no app store. The file is the whole app, and it belongs to you.

Your changes stick

Jot a note, check a box, drag things around. It saves itself right back into the file. Reopen it next week and it's exactly as you left it.

Quiet and private

It runs entirely on your computer. No sign-up, no tracking, nothing leaves your machine. It even works on a plane.

How it works

Install HTML Clay

One small, free app. One time.

Double-click a file

Any .htmlclay file opens in its own clean window. No browser tabs, no clutter.

Just use it

Type, edit, organize. It saves automatically, back into the same file.

Is it safe?

As safe as a web page, because that's all it is. It runs in your browser's sandbox, can't reach into your computer, and the only file it can change is its own.

The one real risk is the same as anywhere online: be cautious if a page asks you to download and run a separate program.
